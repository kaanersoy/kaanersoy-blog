## 📦 Installation

#### 📰 Clone the project

```bash
git clone https://github.com/kaanersoy/kaanersoy-blog.git
```

#### 🔻 Install the depedencies

```bash
yarn install
```

#### 🏃‍♂️ Start the project

```bash
# for dev
yarn dev
```
